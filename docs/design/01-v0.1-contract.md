# v0.1 Contract

The first real package target is a simple univariate Gaussian animal model.

## Planned Julia Capability

Phase 1 should support:

- pedigree validation and sorting;
- ID recoding;
- founder and unknown-parent handling;
- direct sparse `Ainv`;
- fixed-effect design;
- random-effect design;
- REML/ML fitting;
- EBVs/BLUPs;
- variance components;
- heritability.

## Planned R Surface

The R twin should eventually parse:

```r
hsquared(
  y ~ sex + age + animal(1 | id, pedigree = ped),
  data = dat,
  family = gaussian()
)
```

Phase 0 must not make this executable. It is a contract target.

## Engine Boundary

Julia owns sparse relationship construction and fitting. R owns formula
capture, data validation, user-facing messages, S3 methods, and extractors.

## Gate

v0.1 requires a Mrode simple animal-model example, hand-checked tiny pedigree
tests, and at least one comparator check where available.
