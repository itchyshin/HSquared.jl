## Implementation

`centered_markers` returns `(W, p, k)` with `W = M − 2p`, `k = 2Σp(1−p)`, so
`G == W·Wᵀ/k`. `genomic_relationship_matrix` now delegates to it (one source of
the centering/scaling; the existing 9-check G testset guards the refactor).
`fit_snp_blup` treats `W` as the random-effect design with an identity prior and
per-marker variance `σ²_g/k`, solving through the existing `henderson_mme`. It
returns `marker_effects = â` and `gebv = W·â` — relabelled because on a SNP-BLUP
spec the random effects are marker effects, not animal breeding values (reusing
the EBV/`breeding_values` vocabulary there would mislabel them).

## The Equivalence (and why the naive route fails)

A VanRaden `G` is always singular (centering ⇒ `1∈null(G)`), so the equivalence
**cannot** be checked by inverting `G`. The test routes the GBLUP side through
the marginal `V = σ²_g·G + σ²_e·I`: `û = σ²_g·G·V⁻¹·(y − Xβ̂)`. Then
`û == fit.gebv = W·â` to machine precision (~5e-17 for `n>m`, ~2e-16 for `n<m`).
An earlier attempt to compare against `Ginv`-based GBLUP only matched in the
ridge→0 limit (~3e-2 at ridge = 1e-2) and is explicitly avoided.

## Public Claim Audit

Allowed: SNP-BLUP/RR-BLUP supplied-variance marker-effect solve (experimental);
`gebv = W·â` equals GBLUP (self-consistency / equivalence proof).

Blocked: claiming the equivalence proves field-correct absolute numbers (it only
proves the parameterization bridge — external comparator still gates that);
weighted/standardized/Bayesian marker priors (identity prior only); REML σ²_g
estimation; any performance claim for large `m`.

## Tests Of The Tests

The pinned `p`/`k`/β/marker-effects/GEBV are hand-reproducible anchors; the
centering check (`colsums(W)=0`) guards the centering; the equivalence via the
marginal `V` is an independent algebraic route, so agreement to ~1e-16 across
both `n>m` and `n<m` is strong evidence the convention is internally consistent.
